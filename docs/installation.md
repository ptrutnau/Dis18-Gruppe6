# Installation

## Requirements

This project requires Python 3.10+.

To install the necessary libraries, create a `requirements.txt` file with the following contents:

```plaintext
requests==2.26.0
pywikibot==6.7.0
pandas==1.3.3
openpyxl==3.0.9
python-dotenv==0.19.2
mkdocs==1.2.3
mkdocs-material==8.1.2
